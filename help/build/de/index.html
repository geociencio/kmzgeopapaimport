Benutzerhandbuch: KMZ Geopapa Import (v0.2)¶

Dieses QGIS-Plugin ermöglicht den Import von Daten aus .kmz-Dateien, die von der Geopaparazzi-Anwendung generiert wurden, und konvertiert sie in CSV-Dateien zur Verwendung in QGIS oder anderer Software.

Import-Schritte¶

  1. Öffnen Sie das Plugin:

    Gehen Sie in QGIS zum Menü „Erweiterungen“ (Plugins) und wählen Sie „Import kmz files“, um den Importdialog zu öffnen.

  2. KMZ-Datei auswählen:

    Klicken Sie auf die Schaltfläche „…“ neben dem Feld „KMZ-Datei“, um die .kmz-Datei auszuwählen, die Sie aus Geopaparazzi exportiert haben.

  3. Formular auswählen:

    Das Plugin analysiert die KMZ-Datei und listet alle Formulare (Layer) auf, die es im Dropdown-Menü „Formular zur Verarbeitung auswählen“ findet. Wählen Sie das Formular aus, das Sie importieren möchten.

  4. Ausgabeordner auswählen:

    Klicken Sie auf die Schaltfläche „…“ neben „Ausgabeordner“, um den Ordner auszuwählen, in dem die resultierende CSV-Datei gespeichert werden soll.

  5. (Optional) Koordinaten reprojizieren:

    Wenn Sie die ursprünglichen Koordinaten (WGS84) in ein anderes Bezugssystem konvertieren möchten:

    • Aktivieren Sie das Kontrollkästchen „In Projekt-CRS reprojizieren“.

    • Wählen Sie das Ziel-CRS aus dem Dropdown-Menü.

    • Die Ausgabe-CSV-Datei enthält zwei neue Spalten, xtrsf und ytrsf, mit den reprojizierten Koordinaten.

  6. Verarbeiten:

    Klicken Sie auf die Schaltfläche „Verarbeiten“. Das Plugin erstellt eine .csv-Datei imAusgabeordner. Der Dateiname basiert auf dem Namen des ausgewählten Formulars (z. B. mein_formular.csv).

Ergebnis¶

Nach Abschluss finden Sie im Ausgabeordner eine CSV-Datei, die bereit ist, in QGIS als getrennte Text-Layer geladen oder in anderer Software analysiert zu werden.
