Guide de l’utilisateur : KMZ Geopapa Import (v0.2)¶

Ce plugin pour QGIS permet d’importer des données depuis des fichiers .kmz générés par l’application Geopaparazzi, en les convertissant en fichiers CSV pour une utilisation dans QGIS ou un autre logiciel.

Étapes d’importation¶

  1. Ouvrir le Plugin :

    Dans QGIS, allez dans le menu « Extensions » (Plugins) et sélectionnez « Import kmz files » pour ouvrir la boîte de dialogue d’importation.

  2. Sélectionner le fichier KMZ :

    Cliquez sur le bouton « … » à côté du champ « Fichier KMZ » et sélectionnez le fichier .kmz que vous avez exporté depuis Geopaparazzi.

  3. Choisir le formulaire :

    Le plugin analysera le fichier KMZ et listera tous les formulaires (couches) qu’il trouvera dans le menu déroulant « Choisir le formulaire à traiter ». Sélectionnez le formulaire que vous souhaitez importer.

  4. Sélectionner le dossier de sortie :

    Cliquez sur le bouton « … » à côté de « Dossier de sortie » pour choisir le dossier où le fichier CSV résultant sera enregistré.

  5. (Facultatif) Reprojeter les coordonnées :

    Si vous souhaitez convertir les coordonnées d’origine (WGS84) vers un autre système de référence :

    • Cochez la case « Reprojeter vers le SCR du projet ».

    • Choisissez le SCR de destination dans le menu déroulant.

    • Le fichier CSV de sortie inclura deux nouvelles colonnes, xtrsf et ytrsf, avec les coordonnées reprojetées.

  6. Traiter :

    Cliquez sur le bouton « Traiter ». Le plugin créera un fichier .csv dans le dossier de sortie. Le nom du fichier sera basé sur le nom du formulaire que vous avez sélectionné (par ex. mon_formulaire.csv).

Résultat¶

Une fois terminé, vous trouverez un fichier CSV dans le dossier de sortie prêt à être chargé dans QGIS en tant que couche de texte délimité ou à être analysé dans un autre logiciel.
