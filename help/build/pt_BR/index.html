Guia do Usuário: KMZ Geopapa Import (v0.2)¶

Este plugin para QGIS permite importar dados de arquivos .kmz gerados pelo aplicativo Geopaparazzi, convertendo-os em arquivos CSV para uso no QGIS ou outro software.

Passos para a importação¶

  1. Abrir o Plugin:

    No QGIS, vá ao menu “Complementos” (Plugins) e selecione “Import kmz files” para abrir a caixa de diálogo de importação.

  2. Selecionar Arquivo KMZ:

    Clique no botão “…” ao lado do campo “Arquivo KMZ” e selecione o arquivo .kmz que você exportou do Geopaparazzi.

  3. Escolher o Formulário:

    O plugin analisará o arquivo KMZ e listará todos os formulários (camadas) que encontrar no menu suspenso “Choose the form to process”. Selecione o formulário que deseja importar.

  4. Selecionar Pasta de Saída:

    Clique no botão “…” ao lado de “Pasta de saída” para escolher a pasta onde o arquivo CSV resultante será salvo.

  5. (Opcional) Reprojetar Coordenadas:

    Se você deseja converter as coordenadas originais (WGS84) para outro sistema de referência:

    • Marque a caixa “Reprojetar para o SRC do projeto”.

    • Escolha o SRC de destino no menu suspenso.

    • O arquivo CSV de saída incluirá duas novas colunas, xtrsf e ytrsf, com as coordenadas reprojetadas.

  6. Processar:

    Clique no botão “Processar”. O plugin criará um arquivo .csv na pasta de saída. O nome do arquivo será baseado no nome do formulário que você selecionou (ex. meu_formulario.csv).

Resultado¶

Ao finalizar, você encontrará um arquivo CSV na pasta de saída pronto para ser carregado no QGIS como uma camada de texto delimitado ou para ser analisado em outro software.
